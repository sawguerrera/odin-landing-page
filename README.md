# odin-landing-page

Notes summary from project

Using the lessons learned from previous exercises as well as googling styles or properties I didn't completely remember, I found
this to be of good use of how to display things properly on a webpage.

I initially used photos for the main section, but found they overflowed the webpage to far wide, so selected simple icons for the basis of this exercise.

The sign up form took a little thought before actioning as I had to make sure I was using the right properties and understand how things are laid out on a webpage. I also containerized extra divs to ensure flex properties could work.

One thing I could definitely improve upon is working out a way to apply the right level of padding for the entire width of the webpage, rather than doing so section by section. I think there's most likely a better way to ensure the body is responsive, but understood this task as responsiveness not being required.
